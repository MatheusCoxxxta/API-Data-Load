# TypeScript Node Boilerplate
<br/>

## Contains in this boilerplate

- Controllers
- Database
- Models
- Types
- Tests
  - Integration

<br/>

# Database Info

**Client: SQLite3**

<br />

# How to use

### Download

`git clone https://github.com/MatheusCoxxxta/TypeScript-Node-Boilerplate.git server`
<br/>

### Install

` cd server`

`yarn install`
<br/>

### Run

`yarn start`
<br/>

### Test

`yarn test`
